# SimpleTest
Тестовые задания


Схема БД контактов сотрудников состоит из следующих таблиц.

*Отделы departments*
| Поле       | Значение         | Тип        |
|:----------:|:----------------:|:----------:|
| dept_id    | код отдела       | number     |
| dept_name  | название отдела  | varchar2   |


*Сотрудники employers*
| Поле         | Значение                          | Тип        |
|:------------:|:---------------------------------:|:----------:|
| emp_id       | код сотрудника                    | number     |
| emp_name     | ФИО                               | varchar2   |
| dept_id      | код отдела из таблицы departments | number     |
| birth_date   | дата рождения                     | date       |

*Контакты contacts*
| Поле         | Значение                            | Тип        |
|:------------:|:-----------------------------------:|:----------:|
| emp_id       | код сотрудника из таблицы employers | number     |
| cont_type    | тип контакта                        | char       |
|              | e – e-mail                          |            |
|              | p – городской телефон               |            |
|              | m – мобильный телефон               |            |
| cont_text    | значение контакта                   | varchar2   |


Задача
Вывести отчет в виде

| Поле                              | Значение                |
|:---------------------------------:|:-----------------------:|
| Отдел 1                           |                         |
| Сотрудник 1 отдела 1              | Телефоны, e-mail        |
| Сотрудник 2 отдела 1              | Телефоны, e-mail        |
| Отдел 2                           |                         |
| Сотрудник 1 отдела 2              | Телефоны, e-mail        |
| Сотрудник 2 отдела 2              | Телефоны, e-mail        |
| Отделы, в которых нет сотрудников |                         |
|                                   | Отдел без сотрудников 3 |

…



Программирование

Задача 1
Написать фрагмент программы, вычисляющий сумму положительных элементов двухмерного массива.

Задача 2
Написать фрагмент программы, вычисляющий сумму элементов двухмерного массива, исключая максимальный и минимальный.

Задача 3
Написать рекурсивную процедуру вычисления факториала.

Задача 4
Написать фрагмент программы, перемножающий две матрицы.

Задача 5
Написать фрагмент программы, вычисляющий детерминант матрицы 3 Х 3.

Задача 6
Написать фрагмент кода, обрабатывающий поле для игры в тетрис размерами 20 на 10 после очередного хода. Ход начинается с появления фигуры, включает её повороты и сдвиги, выполняемые игроком и завершается, когда фигура зафиксировалась на игровом поле.
